# 브랜딩/locales/README 일반화 (spec T6, ADR-0004)

이름·설명을 전 사이트·다통화에 맞게 일반화. **gecko id 고정**(변경 금지 — AMO 동일 앱).

## 변경
- `_locales/ko/messages.json` · `_locales/en/messages.json`:
  - `extensionName` — 예 ko "환율 호버 → 원화" / en "Currency → KRW Hover".
  - `extensionDescription` — 어떤 사이트/통화든 원화 환산, **"사용자가 켠 사이트에서만 동작"** 명시(all_urls 권한 정당화·AMO 심사 대비).
  - `actionTitle` — 일반화.
- `README.md`: 제목·기능·구조(신규 파일 currency/rates/sites)·권한 표·사용법 갱신.

## acceptance
- `npx web-ext lint` error 0.
- gecko id `ali-usd-krw-hover@local` 불변 확인.
- 이름/설명에 AliExpress·USD 고정 표현 제거, per-site 동작 문구 포함.
