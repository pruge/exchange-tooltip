# 개인정보 처리방침 (Privacy Policy)

**최종 업데이트: 2026-06-15**

이 확장 프로그램(AliExpress USD → KRW Hover)은 사용자 데이터를 수집·저장·전송하지 않습니다.

## 1. 수집하는 정보

**없음.**

이 확장은 어떤 개인 식별 정보, 브라우징 활동, 페이지 내용, 클릭 이력, 위치 정보, 계정 정보도 수집하지 않습니다.

## 2. 외부 통신

환율 정보를 가져오기 위해 다음 공개 API 에 요청을 보냅니다:

- `https://open.er-api.com/v6/latest/USD`

이 요청에는 **사용자 정보가 일절 포함되지 않습니다** — 단순히 USD 기준 환율 데이터를 조회할 뿐입니다. 요청 본문이 없고, 인증 토큰도 사용하지 않으며, 표준 브라우저 User-Agent 외에는 어떤 추가 헤더도 전송하지 않습니다.

## 3. 로컬 저장

다음 정보가 브라우저 로컬 storage (`browser.storage.local`) 에 저장됩니다. **외부 전송 없음.**

| 키 | 내용 | 용도 |
|---|---|---|
| `usdKrwRate` | 환율 값 + 조회 시각 | 1시간 캐시로 API 호출 최소화 |
| `usdKrwPrefs` | 선택한 테마, 환율 정보 표시 여부 | 사용자 옵션 저장 |

확장을 제거하면 위 데이터도 함께 제거됩니다.

## 4. 페이지 접근

알리익스프레스(`*.aliexpress.com`, `*.aliexpress.us`) 페이지의 가격 텍스트(예: `$12.34`)를 읽어 원화 환산값을 툴팁으로 표시합니다.

- 읽은 가격 정보는 **로컬에서만 처리**되며 외부로 전송되지 않습니다
- 가격 외 페이지 내용(상품 이름, 리뷰, 사용자 정보 등)은 읽지 않습니다
- 페이지 DOM 을 수정해 가격 위에 호버 영역을 표시하지만, 페이지 자체에 영구적 변경을 가하지 않습니다

## 5. 제3자 서비스

- **open.er-api.com** — 무료 공개 환율 API. 우리는 어떤 사용자 식별 정보도 보내지 않으며, 이 서비스는 사용자 정보를 받지 않습니다. 자세한 사항은 [open.er-api.com 정책](https://www.exchangerate-api.com/terms) 참조.

## 6. 쿠키 / 트래킹

쿠키, 픽셀, 분석 도구, 광고 식별자 등을 일절 사용하지 않습니다.
